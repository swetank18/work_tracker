# Work Tracker Dashboard

A lightweight dashboard app to track work across:
- Research
- Projects
- Startup
- College Curriculum
- Gym
- Club Work

## Run locally

Because this is a static app, open `index.html` directly in your browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Features

- Add items with area, priority, due date, and notes
- Mark work as done/open
- Filter items by area
- Clear all completed work in one click
- Dashboard stats and completion rate
- Persists data in browser `localStorage`
